# Get / set store from React component
## useStore

```ts
const { useStore } = cervello({ example: true })
```

## Description
React hook that allows you to have a reactive store which re-renders when a new value was set

## Return value

| Type    |default value         |Is reactive?  |
|:---     |:---             |---	        |
| **Object**  | _initialValue_  | Yes, you can modify a property and it will notify all the components using `useStore` to be re-render   |

:::caution
If you want to modify a property of the store, you need to use it without destructuring, like this:
```tsx
const { useStore } = cervello({ isActive: true })

const ExampleComponent = () => {
  const store = useStore()
  return (
    <button onClick={() => store.isActive = false}>
      Set Active to false
    </button>
  )
}
```
:::
